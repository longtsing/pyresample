教程
=========

下面的教程通过一系列 pyresample 提供的功能，并考虑到更大的总体目标。这些教程包括创建或加载数据，以提供使用 pyresample 的端到端示例，而无需用户进行任何操作。教程还假设对 pyresample 的先验知识或经验非常少，用户应该能够跟随它们，而无需对主题有深入的了解。因此，教程通常会避免提供有关所涉及概念的大量详细信息，而只是说明正在做什么。教程与 :doc:`操作指南 <../howtos/index>` 的不同之处在于操作指南专注于单个功能或任务，并假设对正在执行的任务有一定的基础知识水平。
